Welcome to the Birth Symbols API

Use these routes to request data on birth month and birth place symbols.

Routes What they do
/api/ get data on birth stones and flowers for all months and
national colors, animal, and fruit for all countries
/api/month/:month get data on a birth stone and flower for specified month
/api/country/:country get data on national colors, animal, and fruit for specified country
API created by Bianca. Visit this project's GitHub repo to see the code.
