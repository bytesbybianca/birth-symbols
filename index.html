Welcome to the Birth Symbols API

Use these routes to request data on birth month and place symbols.

/api/ to view birth stones and flowers for all months and
national colors, animal, and fruit for all countries
/api/month/:month to view a birth stone and flower for specified month
/api/country/:country to view national colors, animal, and fruit for specified country
API created by Bianca. Visit this project's GitHub repo to see the code.
